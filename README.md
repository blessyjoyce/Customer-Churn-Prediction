# Customer Churn Prediction System

## Overview

The Customer Churn Prediction System is a Machine Learning-based application designed to predict whether a customer is likely to discontinue a service or leave a company. By analyzing customer demographics, account information, and service usage patterns, the system helps businesses identify at-risk customers and take proactive retention measures.

## Problem Statement

Customer retention is a critical challenge for businesses. Acquiring new customers is often more expensive than retaining existing ones. This project aims to predict customer churn using machine learning techniques, enabling organizations to improve customer satisfaction and reduce revenue loss.

## Dataset Features

The model utilizes customer-related information such as:

- Customer ID
- Gender
- Senior Citizen Status
- Partner Status
- Dependents
- Tenure
- Phone Service
- Internet Service
- Contract Type
- Payment Method
- Monthly Charges
- Total Charges
- Churn Status (Target Variable)

### 1. Data Collection

### 2. Data Preprocessing
- Handling missing and inconsistent values.
- Encoding categorical variables.

### 3. Exploratory Data Analysis (EDA)

- Service usage and contract type analysis.

### 4. Model Development
Machine Learning algorithms were used to build the prediction model:

- Logistic Regression

- Random Forest Classifier
- Other classification techniques (if applicable)

### 5. Model Training
- Model trained using historical customer data.
- Hyperparameter tuning performed to improve performance.
- Cross-validation used to ensure model reliability.

### 6. Model Evaluation
The trained model was evaluated using:

- Accuracy Score
- Precision
- Recall
- F1-Score
- Confusion Matrix

### 7. Prediction System
- Accepts customer information as input.
- Predicts whether a customer is likely to churn or remain with the company.
- Assists businesses in implementing targeted retention strategies.
